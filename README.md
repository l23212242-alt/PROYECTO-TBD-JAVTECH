# PROYECTO-TBD-JAVTECH
Desarrollé esta página web, JavTech, para servir como adición y ayuda para mi negocio en línea de venta de computadoras por Facebook. La página consiste en un cátalogo en línea y en un futuro, como método de comunicación entre cliente y vendedor, le permite al cliente donocer mejor la marca con un Index organizado y moderno, y puede visualizar fotografías de los equipos en venta, conocer sus componentes y visualizar el estado en el que se encuentran las computadoras: 1. Disponible, 2. Apartada, 3. Vendida.
